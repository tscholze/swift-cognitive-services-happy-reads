What is it? Happy Read is an iOS app that uses the English feed of the great Deutsche Welle news feed. It enriches the feed articles with a happiness score provided by Microsoft Cognitive Services. That means, that the user is able to choose wether the user want to read happy or unhappy articles. False positive articles are possbile.

Why? This app is targeted to work as a play ground for the Microsoft Cognitive and (in future) other cloud-based services. There is no commercial purpose behind this usecase.
